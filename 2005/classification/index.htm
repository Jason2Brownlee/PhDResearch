Classification Algorithms in WEKA

Overview

This page provides implementation for a number of neural network and immune system based classification algorithms for the WEKA (Waikato Environment for Knowledge Analysis) machine learning workbench. The WEKA platform was selected for the implementation of the selected algorithms because I think its an excellent piece of free software. All comments and bug reports can be directed to me.

Software

The following provides a list of implemented Algorithms

  • Learning Vector Quantization (LVQ)
    • as described in "LVQ_PAK: The Learning Vector Quantization Program Package" by Kohonen, Hynninen, Kangas, Laaksonnen and Torkkola (1996)
    • Provides the following implementations: LVQ1, OLVQ1, LVQ2.1, LVQ3, OLVQ3, Multiple-pass LVQ, Hierarchical LVQ
  • Self-Organizing Map (SOM)
    • as described in "SOM_PAK: The Self-Organizing Map Program Package" by Kohonen, Hynninen, Kangas, and Laaksonnen (1996)
    • Provides the following implementations: SOM, LVQ-SOM, Multiple-pass SOM
  • Feed-Forward Artificial Neural Network (FF-ANN)
    • as described in "Neural Smithing - Supervised Learning in Feedfoward Artificial Neural Networks" by Reed and Mark  (1999)
    • Provides the following implementations: Perceptron, Widrow Hoff, Back Propagation, Bold Driver Back Propagation (Vogl's Method)
  • Artificial Immune Recognition System (AIRS)
    • as described in "Artificial Immune Recognition System (AIRS): An Immune-Inspired Supervised Learning Algorithm" by Watkins, Timmis and Boggess (2004) and "Exploiting Parallelism Inherent in AIRS, an Artificial Immune Classifier" by Watkins and Timmis (2004)
    • Provides the following implementations: AIRS1, AIRS2, Parallel AIRS2
    • A description of this implementation is provided in a technical report "Artificial Immune Recognition System (AIRS) - A Review and Analysis" by Brownlee (2005)
  • Clonal Selection Algorithm (CLONALG)
    • as described in "The clonal selection algorithm with engineering applications" by de Castro and Von Zuben (2000) and "Learning and optimization using the clonal selection principle" by de Castro and Von Zuben (2002)
    • Provides the following implementations: CLONALG and a new CSCA algorithm
    • A description of this implementation is provided in a technical report "The Clonal Section Theory & CLONALG - The Clonal Selection Classification Algorithm (CSCA)" by Brownlee (2005)
  • Immunos-81
    • as described in "The Immune System as a Model for Pattern Recognition and Classification" by Carter (2000)
    • Provides the following implementations: Immunos-81 and a new Immunos-1 and Immunos-2 algorithms
    • A description of this implementation is provided in a technical report "Immunos-81 - The misunderstood artificial immune system" by Brownlee (2005)

Download

The software requires Java 1.5 (5.0) to run, as well as the WEKA library. Download the binary, unzip into the directory and use the provided bat file to start, or use WEKA as per normal ensuring my provided jar file containing the algorithms precedes the WEKA jar file in the Java classpath.
